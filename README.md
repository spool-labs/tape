  **Tapedrive v1 — proof-of-work testnet, retired.**

  This branch preserves the original proof-of-concept that ran on testnet
  through 2025. It is kept here for historical reference and for the
  operators who participated.

  The v2 rewrite is a separate codebase with materially different design
  decisions (proof-of-stake consensus, adaptive erasure coding, chain-native
  writes via Tape Replay). It is not a continuation of this branch.
